# @algebroid/cli

Command line utility for calculating math expressions.
- supports large numbers.
- supported operators: `+` `-` `*` `/` `%` `^`

## Usage

### Install
`npm i -g @algebroid/cli`
### Run
```
# check version
@algebroid/cli --version

# calculate
@algebroid/cli "12 ^ 12 + 4 - ( 80 % 43 )"
```
